## 线 A —「唐小棠的面具」

**核心主题：** 信任与背叛。一个天真少女在最后一刻撕下面具，比任何狼都可怕。

**模拟来源：** 轮32（堕仙队覆灭 + 隐狼觉醒1杀 + 镇煞阵亡）

**最终存活：** 江白、方衡、沈慎、郑守山、唐小棠（5人 / 好人4 : 狼1）

**情绪基调：** 前半段是好人逐步反杀的爽文节奏，D6 突然急转直下，D7 结尾是冰冷的恐怖。

---

### D1 — 抵达

大巴在暴风雪中抵达落雁山庄。12人入住。

老郑（司机）热情地帮大家搬行李，顺便介绍山庄布局。方衡主动亮明身份："我是警察，在市局工作。"——大家稍微安心了。

陈默注意到唐小棠在看他。很快，她就低下头去了。

夜晚，所有人回到各自房间。一切平静。

### D2 — 第一滴血

清晨，陈默没有死。但他感觉不对——昨晚做了梦，梦里的画面模糊不清。

白天，方衡私下找到陈默："你注意到了吗？沈慎那个人……我总觉得他不对劲。"陈默还没来得及回答，叶知秋匆匆跑来："有人……昨晚差点死了。我救了。"她救的是陈默——赵明城夜里闯入陈默房间，被叶知秋的"还魂"拉了回来。但叶知秋不说是谁杀的，她只说"有人差点死了"。

林小满守在苏晚门口一整夜。沈慎在走廊里做了一件谁都不知道的事——他和陈默"交换"了。那天晚上，杀手的刀落在了沈慎的房间——但因为交换，实际没有人受伤。

**夜杀：** 赵明城潜入陈默房间 → 叶知秋还魂救回陈默（用掉1次还魂）
**技能发动：** 沈慎·幻真（交换沈慎↔陈默） | 叶知秋·渡君（还魂陈默） | 林小满·镇煞（守卫苏晚）
**方衡·昭判：** 验沈慎 = 中立，分享给信任的人

### D3 — 女巫的代价

顾言夜里杀了林小满。林小满倒在自己房间门口，手里还攥着画板。

叶知秋感知到死亡，冲过去——又一次还魂。林小满活了，但叶知秋的脸色苍白得可怕。她已经用了两次还魂中的两次。

白天，方衡再次查验——赵明城 = 恶意。他把这个结果告诉了信任的人。林小满醒来后，二话不说，拉着赵明城要"单挑"。镇煞决斗发动——赵明城死了。

但赵明城不是周阳，没有同归。他就这样被林小满杀了。所有人震惊。

**夜杀：** 顾言杀林小满 → 叶知秋还魂救回（用掉第2次还魂，还魂次数耗尽）
**白天：** 林小满·镇煞决斗赵明城 = 杀狼
**方衡·昭判：** 验赵明城 = 恶意，分享

### D4 — 女巫之死

顾言夜里闯入叶知秋的房间。叶知秋感知到了，但她的还魂已经用完了。她只能束手就擒。

叶知秋死了。她的表情很平静，像是早就知道这一天会来。

白天，方衡查验周阳 = 恶意。气氛紧绷到了极点——三狼已死一个，还剩两个。

**夜杀：** 顾言杀叶知秋 ✓死
**方衡·昭判：** 验周阳 = 恶意，分享

### D5 — 同归于尽

周阳夜里杀了苏晚。苏晚倒在血泊中，再也没有醒来。

白天，林小满疯了。她不等方衡的情报，直接冲向周阳——镇煞决斗。但周阳是狼王，同归于尽。林小满和周阳一起倒下。

陈默抱着苏晚的尸体，什么也说不出来。

江白在角落里发现了什么——他的简易陷阱被触发过，他提出质疑顾言，但投票没有通过。

**夜杀：** 周阳杀苏晚 ✓死
**白天：** 林小满·镇煞决斗周阳 = 同归狼王（林小满自杀）
**流放：** 江白质疑顾言（2赞:4反:3弃→保留）

### D6 — 最后的平静

夜晚，没有人死。方衡查验顾言 = 恶意。他把所有情报分享了出来。

但顾言先发制人——他在白天的讨论中质疑方衡，指控方衡是"内鬼"。投票通过，方衡被流放到地下室。

方衡被关进去之前，看了陈默一眼。那个眼神里有很多东西。

**夜杀：** 平安夜
**方衡·昭判：** 验顾言 = 恶意，分享全部
**流放：** 顾言质疑方衡（4赞:0反:2弃→流放）

### D7 — 面具脱落

清晨。陈默、江白、沈慎、郑守山、唐小棠——五个人坐在餐厅里。

气氛很安静。陈默在笔记本上写写画画，试图理清这六天的一切。

然后唐小棠站起来了。

她笑了。不是以前那种害羞的、低下头的笑。是一种……完全不同的笑。

"陈默学长，"她说，声音不再小了，"你查了那么多人，有没有查过我？"

方衡昨天被关进地下室之前查的最后一个人是陈默——中立。他没有查过唐小棠。从来没有。因为谁会怀疑一个可爱的、喜欢他的学妹呢？

唐小棠的手指划过桌面。"你们把三只狼都解决了。赵明城、周阳、顾言。很厉害。"她的语气像在夸奖孩子。"但是……你们忘了一件事。"

"狼不是只有三只。"

陈默的血凉了。

唐小棠杀了陈默。这是隐狼觉醒后的第一杀——她的能力在三狼全部失去行动能力后解锁。

方衡在地下室里听到了动静。他还有一发子弹。他推开门——朝顾言开了一枪。顾言死了。但方衡指认的是顾言，不是唐小棠。

因为方衡从头到尾，就没有怀疑过唐小棠。

沈慎坐在角落里，面无表情地看着这一切。他又活过了一天。

**夜杀（隐狼觉醒）：** 唐小棠杀陈默 ✓死
**方衡·昭判枪：** 枪杀顾言（但真凶是唐小棠）
**最终存活：** 江白、方衡、沈慎、郑守山、唐小棠

### 尾声

第七天结束了。轮回没有结束。

陈默在下一个 Day1 醒来。他记得一切。

他记得唐小棠最后那个笑容。

## 线 B —「银水的代价」

**核心主题：** 善意被利用。一个温柔的人为了保护自己救过的人，赌上了自己的命。

**模拟来源：** 轮46（自刀骗药 + 渡君被骗复活）

**最终存活：** 陈默、苏晚、江白、顾言、唐小棠、赵明城（6人 / 好人3 : 狼3）

**情绪基调：** 全程压抑。好人的善意一步步被狼人利用，最终被反噬。

---

### D1 — 抵达

同上。方衡亮明警察身份。大家入住。

### D2 — 自刀骗局

第一夜。平安夜——没有人死。

清晨，周阳倒在走廊上，浑身是"血"。叶知秋冲过去——她感知到了死亡的气息。周阳的脉搏微弱，伤口看起来触目惊心。

叶知秋犹豫了一秒。她的还魂只有两次。但她不可能见死不救。

她用了。周阳活了。

周阳睁开眼的那一刻，眼里闪过一丝极快的、旁人捕捉不到的东西——那是计谋得逞的得意。但很快就被他招牌式的阳光笑容盖过去了。

"谢……谢谢叶姐。"他虚弱地说。

没有人知道周阳的伤是自己捅的。那把刀的角度、深度，都经过了精心设计——刚好像被堕仙袭击的样子。叶知秋的入殓师经验让她能分辨伤口，但这一次，她没有仔细看。因为她太忙着救人了。

方衡这天晚上查验了周阳 = 恶意。他把结果分享给了信任的人。但周阳刚被"救活"，大家对方衡的情报半信半疑——一个差点死掉的人，怎么可能是坏人？

沈慎和陈默交换了身份。林小满守卫了苏晚。

**自刀骗药：** 周阳自刀 → 叶知秋被骗还魂（浪费1次还魂）
**方衡·昭判：** 验周阳 = 恶意，分享
**技能发动：** 沈慎·幻真（交换沈慎↔陈默） | 林小满·镇煞（守卫苏晚）

### D3 — 连环杀

顾言夜里杀了苏晚。但叶知秋又一次还魂——她不忍心看苏晚死。苏晚是医学院的学生，和叶知秋有一种镜像般的关系：一个在读，一个退学。

叶知秋用了最后一次还魂。她现在只有缚魂了。

白天，林小满发现苏晚的伤口，暴怒。她直接冲向周阳——镇煞决斗。同归于尽。林小满和周阳一起倒下。

陈默在角落里看着这一切，手里的笔记本掉在地上。苏晚刚被救活，林小满就……

叶知秋站在走廊尽头，脸色惨白。她救了两个人，但林小满还是死了。

**夜杀：** 顾言杀苏晚 → 叶知秋还魂救回（用掉第2次还魂，还魂耗尽）
**白天：** 林小满·镇煞决斗周阳 = 同归狼王（林小满自杀）
**方衡·昭判：** 验沈慎 = 友善，分享

### D4 — 消失的尸体

赵明城夜里杀了老郑。但赵明城是清道夫——他抹除了尸体。

第二天早上，老郑不见了。他的房间空了，床铺整齐，像是从来没有住过人。

"老郑呢？"江白问。

没有人知道。或者说，只有赵明城知道。

江白在走廊尽头发现了老郑常戴的那顶旧毛线帽，掉在角落里。他开始怀疑顾言，提出质疑——但投票没有通过。

**夜杀：** 赵明城杀老郑 ✓死 {抹尸}
**流放：** 江白质疑顾言（2赞:4反:3弃→保留）
**方衡·昭判：** 验江白 = 友善，分享

### D5 — 方衡之死

赵明城夜里杀了方衡。同样，抹尸。

方衡消失了。他留在房间里的只有那把没有用过的手枪——被赵明城拿走了。

方衡最后查验的是苏晚 = 友善。这个情报再也没有机会分享出去了。

**夜杀：** 赵明城杀方衡 ✓死 {抹尸}
**方衡·昭判：** 验苏晚 = 友善（未分享）

### D6 — 叶知秋之死

赵明城杀了叶知秋。抹尸。

叶知秋消失了。没有尸体，没有痕迹。一个温柔的人就这样从这个世界上被"擦掉"了。

她救过三个人——陈默、周阳（被骗）、苏晚。其中一个是狼。她的善意，被利用了。

**夜杀：** 赵明城杀叶知秋 ✓死 {抹尸}

### D7 — 最后的冬天

清晨。陈默、苏晚、江白、沈慎、顾言、唐小棠、赵明城——七个人坐在餐厅里。

六天里，五个人消失了。老郑、方衡、叶知秋——没有尸体，没有告别。林小满和周阳在走廊上同归于尽，血迹已经被擦干净了。

陈默翻开笔记本。他记下了所有线索：
- 方衡分享过"周阳 = 恶意"。但周阳和林小满一起死了。
- 方衡验过沈慎 = 友善、江白 = 友善、苏晚 = 友善。
- 叶知秋在D2救了一个"差点死掉"的人——周阳。但周阳是坏人。
- 老郑的帽子在走廊尽头。他的房间在204，隔壁203是顾言。
- 赵明城一直没有被查验过。他来的理由是"商务旅行"。他和方衡似乎认识。

沈慎坐在角落里，面无表情。他又活过了一天。

赵明城微笑着给大家倒了茶。"暴风雪总会停的，"他说，"我们只需要等。"

第七天结束了。轮回继续。

陈默在下一个 Day1 醒来。他记得一切。尤其是叶知秋救周阳时那个毫不犹豫的动作。

**最终存活：** 陈默、苏晚、江白、顾言、唐小棠、赵明城

### 尾声

银水——叶知秋救过的人，被称为"银水"。这是信任的标记。

但信任，是可以被伪造的。

## 线 C —「方衡的最后一枪」

**核心主题：** 一个黑警的救赎。他手上有枪、有直觉、有秘密，但他永远差一步。

**模拟来源：** 轮42（镇煞双杀 + 极低死亡2人）

**最终存活：** 陈默、苏晚、江白、方衡、沈慎、郑守山、林小满、周阳、唐小棠（9人 / 好人5 : 狼4）

**情绪基调：** 紧张的推理剧。好人阵营靠方衡的直觉和林小满的决斗逐步压制狼队，但每一步都差一点就翻车。

---

### D1 — 抵达

同上。方衡亮明警察身份。老郑帮大家搬行李。

### D2 — 方衡的第一刀

顾言夜里杀了方衡。

方衡倒在血泊中。叶知秋冲过来——还魂。方衡活了。

这是方衡第一次"死"。他睁开眼，看到叶知秋苍白紧张的脸。她身上有消毒水的味道。

"别告诉别人，"叶知秋低声说。

方衡查验周阳 = 恶意。他把结果分享给了信任的人。

沈慎和陈默交换了身份。林小满守卫了苏晚。

**夜杀：** 顾言杀方衡 → 叶知秋还魂救回（用掉1次还魂）
**方衡·昭判：** 验周阳 = 恶意，分享
**技能发动：** 沈慎·幻真（交换沈慎↔陈默） | 林小满·镇煞（守卫苏晚）

### D3 — 同归于尽

赵明城夜里杀了方衡——又一次。抹尸。

但方衡在"死"之前做了最后一件事：他指认了赵明城。手枪响了一声。赵明城死了。

方衡临死反杀——猜中了真凶。子弹正中赵明城胸口。

赵明城是清道夫狼，不是狼王，没有同归。方衡和赵明城一起倒下。

白天，林小满不等任何人，直接冲向周阳——镇煞决斗。但周阳是狼王。同归于尽。林小满和周阳一起倒下。

**夜杀：** 赵明城杀方衡 ✓死 {抹尸}
**方衡·临死反杀：** 指认赵明城 → 命中！赵明城死
**白天：** 林小满·镇煞决斗周阳 = 同归狼王（林小满自杀）
**方衡·昭判：** 验郑守山 = 友善，分享

### D4 — 老郑之死

顾言夜里杀了老郑。老郑倒在204房间里，手里还攥着那条旧毛巾。

叶知秋感知到了死亡。她冲过去——还魂。老郑活了。这是她第二次还魂，也是最后一次。

白天，江白发现自己的陷阱被触发——有人受伤了。他提出质疑顾言。但投票只有2赞3反，没有通过。

顾言坐在角落里，面无表情。他知道江白在怀疑他，但他什么都不说。

**夜杀：** 顾言杀老郑 → 叶知秋还魂救回（用掉第2次还魂，还魂耗尽）
**流放：** 江白质疑顾言（1赞:3反:3弃→保留）

### D5 — 沉默的走廊

夜晚，没有人死。沈慎和顾言"交换"了——但没有造成误杀。

白天，江白再次质疑顾言。投票1赞3反，没有通过。

这栋楼安静得像坟墓。方衡走了，林小满走了，赵明城和周阳也走了。十二个人，现在只剩八个。

陈默坐在二楼走廊的窗户边，看着外面的暴风雪。他翻开笔记本，写下：
- 方衡查过：周阳=恶意、郑守山=友善
- 方衡临死前杀了赵明城——说明方衡知道赵明城是狼
- 林小满和周阳同归——周阳是狼王
- 三狼已灭（赵明城、周阳，加上……还有一个）
- 顾言被江白质疑了两次，都未通过

**夜杀：** 平安夜
**流放：** 江白质疑顾言（1赞:3反:3弃→保留）

### D6 — 漫长的冬天

夜晚，没有人死。沈慎和江白"交换"了——无事发生。

白天，平静。大家都不说话了。

陈默注意到，唐小棠一直坐在角落里，抱着膝盖，一句话也不说。她的眼神有些空洞。

**夜杀：** 平安夜

### D7 — 未完的棋局

清晨。没有人死。

陈默、苏晚、江白、沈慎、叶知秋、郑守山、顾言、唐小棠——八个人坐在餐厅里吃早饭。

暴风雪还在下。

陈默看着顾言。顾言在纸上写公式，头也不抬。

老郑坐在角落里喝粥，他活了下来——因为叶知秋。他说了一句："今晚……我好像听到了楼上有脚步声。从东边楼梯往上的。"

204隔壁是203，顾言的房间。东楼梯往上，是三楼女生层。

方衡不在了。没有人有枪。没有人有直觉。

第七天结束了。轮回继续。

陈默在下一个 Day1 醒来。他记得一切。他知道了两只狼——赵明城和周阳。他还知道，有第三只狼活着。可能是顾言。

但他没有证据。

**最终存活：** 陈默、苏晚、江白、方衡、沈慎、郑守山、林小满、周阳、唐小棠

### 尾声

方衡开过两枪。第一枪打中了赵明城。第二枪——他没有机会开了。

他是个黑警。收过钱，消过案。但在落雁山庄的七天里，他做的每一件事都是对的。

他的直觉从来没有错过。但直觉不等于证据。在这个世界里，没有证据，就没有正义。
