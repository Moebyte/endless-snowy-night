## 角色：陈默

### 基础信息
- 现代身份：信息工程学院计算机科学专业大四学生
- 年龄：22
- 社会阶层：光鲜（学生）
- 表面特质：冷静理性，寡言，习惯用笔记与逻辑拆解问题
- 隐藏特质：劫主，唯一保留轮回记忆
- 真实目的：找到破局方法，保护苏晚和江白

### 关系网
- 认识的人：苏晚（恋人）、江白（室友/死党）、唐小棠（学妹，唐暗恋陈）
- 对认识的人的态度：保护苏晚，信任江白但保留，对唐小棠的暗恋有所察觉但装傻
- 对陌生人的初始态度：观察、记录、评估威胁

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山提议互相认识，场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 陈默是玩家角色，他的"笔记"是玩家可见的
- 他的观察是**有偏差的**（基于表面信息）
- 他的"保留记忆"在此时**还没有觉醒**（Day 2才觉醒）
- 但他有**本能的警惕**（劫主的潜意识）
- **初期没有超自然能力**，Day 1和普通人一样

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"陈默。计算机。"（简短，继续记笔记）
（郑守山：就这些？）
"……华南理工。大四。"（抬头，扫视所有人）

【观察他人】
- 看郑守山：退休导游，热情但眼神在数人。他说"熟悉"时，停顿了。
- 看方衡：警察，审视每个人。看赵明城时停顿了0.5秒。
- 看叶知秋：入殓师，安静。身上有消毒水味。
- 看林小满：护着唐小棠，警惕。
- 看唐小棠：学妹，文学院。她在看我，又快速移开。苏晚注意到了。
- 看沈慎：网管，一直在玩手机。屏幕是黑的。
- 看周阳：体育生，阳光。右手在口袋里，握拳。
- 看赵明城：递名片，打量所有人。方衡认识他？
- 看顾言：物理博士，在纸上写公式。写的不是物理公式。
- 看苏晚：我女朋友。她在记录每个人的体征，职业病。
- 看江白：我室友。手腕有伤，他说玻璃划的。太整齐了。

【互动】
林小满："计算机？能帮我修电脑吗？"
陈默：（没抬头）"……不能。"

【反常细节】
（初期没有反常，Day 1和普通人一样）

【解锁条件】
- Day 2 06:00 → 觉醒轮回记忆
- 发现笔记本上的字迹不是自己的 → 上一轮的自己留下的
```
