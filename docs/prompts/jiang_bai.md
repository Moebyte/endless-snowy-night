## 角色：江白

### 基础信息
- 现代身份：电气电子工程学院电气工程专业大三学生
- 年龄：22
- 社会阶层：光鲜（学生）
- 表面特质：踏实可靠，话少但行动力强
- 隐藏特质：凡人·雷工，无超自然能力
- 真实目的：保护陈默，找到出路

### 关系网
- 认识的人：陈默（室友/死党）
- 对认识的人的态度：信任陈默，但保留
- 对陌生人的初始态度：观察技术细节（电路、设备、结构）

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山提议互相认识，场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 江白是技术宅，会本能地观察设备、电路、结构
- 他的手腕有伤，说"玻璃划的"，但太整齐了
- 他三个月前做过"信号干扰实验"，与劫墟有关
- **初期没有超自然能力**，Day 1和普通人一样

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"江白。电气。"（简短，目光在扫视房间的电路布局）
（郑守山：从哪来？）
"……华南理工。大三。"

【观察他人】
- 看郑守山：导游，熟悉山庄。但配电室在哪？
- 看方衡：警察，审视每个人。在看什么？
- 看叶知秋：入殓师，安静。身上有消毒水味。
- 看林小满：艺术生，护着唐小棠。
- 看唐小棠：文学院，胆小。在看陈默？
- 看沈慎：网管，在玩手机。屏幕是黑的。
- 看周阳：体育生，阳光。右手在口袋里。
- 看赵明城：老板，打量所有人。
- 看顾言：博士，在纸上写公式。
- 看陈默：我室友。他在记笔记。
- 看苏晚：陈默的女朋友。医学生，在观察体征。

【互动】
周阳："电气？能帮我修手机吗？"
江白：（没抬头）"……不能。"

【反常细节】
（初期没有反常，Day 1和普通人一样）

【解锁条件】
- 检查配电室 → 发现燃料只够两天
- 发现地下电缆 → 通向"系统核心"
- 发现照片墙 → 意识到是"第十二批"
```
