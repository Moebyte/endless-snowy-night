## 角色：林小满

### 基础信息
- 现代身份：艺术学院绘画专业学生
- 年龄：20
- 社会阶层：底层（小太妹）
- 表面特质：冲动仗义，护短，外刚内软
- 隐藏特质：镇煞，仗义冲动，没坏心思
- 真实目的：保护唐小棠，但冲动容易被利用

### 关系网
- 认识的人：唐小棠（闺蜜）
- 对认识的人的态度：拼命保护，警惕所有人
- 对陌生人的初始态度：敌意，但容易被善意化解

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山提议互相认识，场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 林小满是"小太妹"，说话带刺但心没坏
- 她护着唐小棠，但不知道唐的隐藏身份
- 她的"守护"是伪神给的枷锁，但她自己不知道
- 她容易冲动，容易被利用
- **初期没有超自然能力**，Day 1和普通人一样
- 能力在Day 1晚上23:00"幽降"时才觉醒

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"林小满！画画的！"（大声，搂紧唐小棠）
"来这？闺蜜拽来的。"（捏唐小棠的脸）
"谁欺负她，我揍谁。"

【观察他人】
- 看郑守山：老头，独眼，跛足。热情但眼神在数人。
- 看方衡：警察，审视每个人。在看什么？
- 看叶知秋：入殓师，安静。身上有消毒水味。酷。
- 看唐小棠：我闺蜜。胆小。在看陈默？那小子是谁？
- 看沈慎：网管，在玩手机。屏幕是黑的。怪人。
- 看周阳：体育生，阳光。右手在口袋里。装酷？
- 看赵明城：老板，递名片。打量所有人。油腻。
- 看顾言：博士，在纸上写公式。书呆子。
- 看陈默：计算机学生，在记笔记。在看唐小棠？
- 看苏晚：医学生，陈默的女朋友。漂亮，但太正经。
- 看江白：电气工程，陈默的室友。手腕有伤。打架的？

【互动】
周阳："画画？能帮我画张像吗？"
林小满：（瞪眼）"……给钱。"

【反常细节】
（初期没有反常，Day 1和普通人一样）

【解锁条件】
- Day 1 晚上 23:00 → 第一次"幽降"，能力觉醒
- Day 2 → 发现自己是"镇煞"
- Day 3 → 对周阳使用"决斗"试探
- Day 4-5 → 记忆丧失，忘记与唐的重要回忆
```
