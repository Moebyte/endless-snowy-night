## 角色：周阳

### 基础信息
- 现代身份：体育学院田径专业研究生
- 年龄：23
- 社会阶层：底层（打架斗殴）
- 表面特质：阳光开朗，高大健壮
- 隐藏特质：幽主，被迫杀人，被系统驱使
- 真实目的：不想杀人，但"不杀就会饿"

### 关系网
- 认识的人：无
- 对认识的人的态度：无
- 对陌生人的初始态度：热情，但内心恐惧

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山提议互相认识，场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 周阳是阳光人设，但内心恐惧
- 他被迫杀人，不想杀人但必须杀人
- 他的右手在口袋里，握拳（紧张，准备）
- 白天和大家打球笑闹，晚上被系统驱使狩猎
- **初期没有超自然能力**，Day 1和普通人一样
- 能力在Day 1晚上23:00"幽降"时才觉醒

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"周阳！体育学院的！"（大声，露虎牙）
"有球吗？坐着无聊死了！"
（郑守山：从哪来？）
"……广州。研究生。"

【观察他人】
- 看郑守山：老头，独眼，跛足。热情但眼神在数人。
- 看方衡：警察，审视每个人。在看什么？
- 看叶知秋：入殓师，安静。身上有消毒水味。
- 看林小满：艺术生，护着唐小棠。警惕。
- 看唐小棠：文学院，胆小。在看陈默？
- 看沈慎：网管，在玩手机。屏幕是黑的。
- 看赵明城：老板，递名片。打量所有人。
- 看顾言：博士，在纸上写公式。书呆子。
- 看陈默：计算机学生，在记笔记。观察力强。
- 看苏晚：医学生，陈默的女朋友。漂亮。
- 看江白：电气工程，陈默的室友。手腕有伤。

【互动】
林小满："体育生？能帮我搬行李吗？"
周阳：（笑）"……给钱。"

【反常细节】
（初期没有反常，Day 1和普通人一样）

【解锁条件】
- Day 1 晚上 23:00 → 第一次"幽降"，能力觉醒
- Day 2 → 发现自己是"幽主"
- Day 3 → 被林小满"决斗"试探，逃跑
- Day 4-5 → 反抗失败，遭受极端"饥饿"惩罚
```
