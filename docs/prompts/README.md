## 使用说明

1. 每个prompt文件包含：
   - 角色基础信息
   - 关系网
   - 场景描述
   - 任务说明
   - 严格约束
   - 输出格式
   - 示例

2. 使用方式：
   - 将prompt文件内容输入AI
   - AI生成该角色的自我介绍和观察
   - 根据解锁条件，逐步揭示隐藏信息

3. 注意事项：
   - 禁止直接说出隐藏身份
   - 禁止内心独白超过1句
   - 必须保留推理空间
