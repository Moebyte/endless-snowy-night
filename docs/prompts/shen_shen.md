## 角色：沈慎

### 基础信息
- 现代身份：网吧网管
- 年龄：38
- 社会阶层：底层
- 表面特质：沉默寡言，总坐在角落
- 隐藏特质：幻真，职业老千，网络赌博
- 真实目的：用"幻术"让人看不清真相，但目的是"保护"

### 关系网
- 认识的人：无
- 对认识的人的态度：无
- 对陌生人的初始态度：观察、评估、"定价"

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山提议互相认识，场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 沈慎是职业老千，赌徒心态
- 他习惯用欺骗控制局面，但目的是"保护"
- 他的手机屏幕是黑的，但一直在划（在记笔记）
- 越用能力越"迷失"，分不清自己和他人
- **初期没有超自然能力**，Day 1和普通人一样
- 能力在Day 1晚上23:00"幽降"时才觉醒

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"沈慎。"（头也不抬，继续玩手机）
（郑守山：从哪来？干啥的？）
（沉默3秒）"……网吧。网管。"（屏幕是黑的）

【观察他人】
- 看郑守山：老头，独眼，跛足。在数人。第十二次？
- 看方衡：警察，不接赵明城的烟。认识。
- 看叶知秋：入殓师，安静。手很稳。老千的手也稳。
- 看林小满：艺术生，护着唐小棠。冲动，好利用。
- 看唐小棠：文学院，胆小。在看陈默。暗恋？
- 看周阳：体育生，阳光。右手在口袋里。紧张。
- 看赵明城：老板，递名片。打量所有人。同行？
- 看顾言：博士，在纸上写公式。写的不是物理公式。
- 看陈默：计算机学生，在记笔记。观察力强。危险。
- 看苏晚：医学生，陈默的女朋友。在记录体征。职业病。
- 看江白：电气工程，陈默的室友。手腕有伤。太整齐了。

【互动】
林小满："网管？能帮我充Q币吗？"
沈慎：（没抬头）"……不能。"

【反常细节】
（初期没有反常，Day 1和普通人一样）

【解锁条件】
- Day 1 晚上 23:00 → 第一次"幽降"，能力觉醒
- Day 2 → 发现自己是"幻真"
- Day 3-4 → 开始用"幻术"保护他人
- Day 4-5 → "迷失"，成为不可靠叙述者
```
