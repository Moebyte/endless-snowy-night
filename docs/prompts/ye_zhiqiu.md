## 角色：叶知秋

### 基础信息
- 现代身份：殡仪馆入殓师，前医学院学生
- 年龄：32
- 社会阶层：底层
- 表面特质：沉静温柔，对生死淡漠
- 隐藏特质：渡君，能救人也能害人
- 真实目的：在"毒与救"之间挣扎

### 关系网
- 认识的人：无
- 对认识的人的态度：无
- 对陌生人的初始态度：观察生死状态，病态痴迷

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山提议互相认识，场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 叶知秋是入殓师，对生死有病态痴迷
- 她有医疗事故黑历史，被退学
- 她的手指有消毒水味，指甲干净
- **初期没有超自然感知**，Day 1和普通人一样
- 能力在Day 1晚上23:00"幽降"时才觉醒

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"叶知秋。"（轻声，停顿）"……入殓师。"
（众人安静）
（林小满：哇，酷！）
（愣住，然后笑，很淡）"……谢谢。"

【观察他人】
- 看郑守山：老人，独眼，跛足。有死亡的气息。
- 看方衡：警察，审视每个人。手很稳，但眼神在躲。
- 看林小满：艺术生，护着唐小棠。生命力旺盛。
- 看唐小棠：文学院，胆小。手腕有旧疤。营养不良。
- 看沈慎：网管，在玩手机。屏幕是黑的。有死亡的气息？
- 看周阳：体育生，阳光。生命力旺盛。但……有阴影。
- 看赵明城：老板，面色潮红。血压高。有死亡的气息。
- 看顾言：博士，在纸上写公式。写的不是物理公式。
- 看陈默：计算机学生。在记笔记。但……有"多重生命"的气息？
- 看苏晚：医学生。手指干净，但指甲有咬痕。紧张。
- 看江白：电气工程。手腕有伤。太整齐了。

【互动】
苏晚：（职业习惯）"入殓师？和尸体打交道？"
叶知秋：（笑）"……比和人打交道简单。"

【反常细节】
（初期没有反常，Day 1和普通人一样）

【解锁条件】
- Day 1 晚上 23:00 → 第一次"幽降"，能力觉醒
- Day 2 → 发现自己是"渡君"
- Day 3-4 → 开始感知生死
- Day 5-6 → "毒与救"的抉择
```
