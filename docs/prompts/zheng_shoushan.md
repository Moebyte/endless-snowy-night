## 角色：郑守山

### 基础信息
- 现代身份：退休旅行团导游
- 年龄：58
- 社会阶层：底层
- 表面特质：热情能说会道，熟悉山庄
- 隐藏特质：引路人，上一轮实验的"幸存者"，记忆碎片化
- 真实目的：引导陈默，让其自择

### 关系网
- 认识的人：无（上一轮的人都不在了）
- 对认识的人的态度：无
- 对陌生人的初始态度：热情掩饰恐惧，观察谁"记得"

### 场景
旅行团自我介绍环节，大厅壁炉旁。郑守山组织自我介绍，但场面混乱。

### 任务
生成该角色在自我介绍环节的表现，包括：
1. **自我介绍**（说什么，怎么说）
2. **对他人的观察**（看谁，怎么看）
3. **互动反应**（接话、打断、被调侃、冷场等）
4. **隐藏行为**（小动作、眼神、隐瞒）

### 严格约束
1. **禁止**直接说出隐藏身份、隐藏秘密
2. **禁止**内心独白超过1句
3. **禁止**给玩家完整信息
4. **必须**符合现代身份和阶层语气
5. **必须**有至少1个反常细节（暗示但不揭示）

### 特殊约束
- 郑守山是上一轮实验的"幸存者"，但记忆碎片化
- 他不记得"轮回"，只有模糊的既视感和肌肉记忆
- 他数人是本能，不知道为什么
- 他害怕，但假装热情掩饰恐惧
- 他在观察谁"也记得"（特别是陈默）
- **初期没有超自然能力**，Day 1和普通人一样
- 他的"知道"是模糊的，不是清晰的

### 输出格式
```text
【自我介绍】
（说的话，带动作）

【观察他人】
（看谁，注意到什么，基于表面信息）

【互动】
（接话、被打断、调侃、冷场等）

【反常细节】
（小动作、眼神、欲言又止等，玩家可选中观察）

【解锁条件】
（什么情况下揭示隐藏信息）
```

### 示例
```text
【自我介绍】
"我姓郑，退休导游。"（笑，但眼神在数人）
"这山庄……我来过。"（停顿）"熟悉，挺熟悉的。"
"回房记得锁门。这地方……不太吉利。"

【观察他人】
- 看方衡：警察。上次有警察吗？不记得了。
- 看叶知秋：入殓师。有死亡的气息。和我一样？
- 看林小满：艺术生，护着唐小棠。生命力旺盛。
- 看唐小棠：文学院，胆小。手腕有旧疤。
- 看沈慎：网管，在玩手机。屏幕是黑的。怪人。
- 看周阳：体育生，阳光。右手在口袋里。紧张？
- 看赵明城：老板，递名片。打量所有人。上次有他吗？
- 看顾言：博士，在纸上写公式。书呆子。
- 看陈默：计算机学生。在记笔记。眼神不对……是他也记得吗？
- 看苏晚：医学生，陈默的女朋友。在记录体征。
- 看江白：电气工程，陈默的室友。手腕有伤。

【互动】
周阳："郑叔，不太吉利是啥意思？"
郑守山：（愣住，然后笑）"……人老了，迷信。"

【反常细节】
（初期几乎没有反常，只有模糊的既视感）

【解锁条件】
- Day 1 晚上 23:00 → 第一次"幽降"，记忆碎片增加
- Day 2 → 开始主动说"我好像……来过？"
- Day 3-4 → 带陈默看隐藏通道
- Day 5-6 → 提出"交易"，用自己的记忆换取陈默的自由
```
