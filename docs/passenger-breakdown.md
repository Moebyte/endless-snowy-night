# 角色来源链条 · 落雁山庄人员构成

> 记录于 2026-06-20 与作者讨论
> 核心原则：这群人不是一个团来的，而是各自有各自的原因，凑巧在同一时间到了同一个地方。

---

## 文学节社团线

**背景：** 全国高校文学节活动，重点大学在邀请名单上。社长拿到名额但临时有事去不了，队长定的出行名单。

| 角色 | 来的原因 | 关系链 |
|------|---------|--------|
| **陈默** | 社团资深社员，被队长拉上凑数 | 核心 |
| **唐小棠** | 文学院学妹，社团成员 | 陈默的学妹 |
| **江白** | 被陈默拉来 | 陈默室友 |
| **苏晚** | 陈默女朋友，正好有空跟着来 | 陈默女友 |
| **林小满** | 被唐小棠拉来 | 唐小棠闺蜜（艺术学院） |

> 社团成员**只有**陈默和唐小棠。其他人都是被拉来的。
