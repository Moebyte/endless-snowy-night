## 📝 使用方法

1. **阅读各角色pool.md**：了解每个角色的"内心独白"、"秘密行动"、"人际互动"
2. **挑选合适片段**：根据主线剧情需要，选择对应的剧情片段
3. **拼接进剧本**：将角色剧情与主线剧情融合，注意信息边界（角色不知道其他角色的隐藏身份）
4. **检查一致性**：确保拼接后的剧情符合角色设定和世界观规则

## ⚠️ 重要提醒

- **信息边界**：角色Agent生成的内容中，有些信息是该角色"不知道"的（如其他角色的隐藏身份）。拼接时需注意，不要让角色"超纲"知道不该知道的事。
- **主线掌控**：陈默（男主）的剧情仍由编剧掌握。角色剧情池只提供"其他角色的视角"，不替代主线。
- **动态调整**：角色剧情池是"静态"的，实际游戏中需要根据玩家选择动态调整。
