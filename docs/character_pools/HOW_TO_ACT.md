# 如何代入角色：从直觉到剧情

> **写给所有 subagent 的指南**
> 当你代入某个角色时，按照这个指南来写剧情。

---

## 核心原则

**你什么都不知道。你只有：**
1. 表面身份（所有人都知道的）
2. 你的秘密（只有你知道的）
3. 你的恐惧（只有你知道的）
4. 对其他人的直觉（模糊的感觉，不知道为什么）

**你绝对不知道：**
- 任何明确的"身份标签"或"能力名称"
- 任何关于"为什么会这样"的解释
- 任何你（角色）不可能知道的信息

---

## 如何把「直觉」扩展成「具体剧情表现」

### 公式：直觉 → 行为/对话/内心独白

| 直觉（they.md里的） | 可以扩展成的具体表现 |
|---------------------|----------------------|
| **"他知道什么"** | - 躲避他的目光<br>- 不自觉地握紧拳头/手插口袋<br>- 内心独白："他看我的眼神……不对劲"<br>- 故意绕开他走 |
| **"同类"** | - 对视时停顿0.5秒<br>- 莫名的熟悉感<br>- 内心独白："她……和我一样？"<br>- 下意识地靠近/避开（取决于关系） |
| **"害怕他"** | - 看到他时身体微微一颤<br>- 说话声音变小/结巴<br>- 内心独白："他能看穿我……"<br>- 找借口离开 |
| **"不干净"** | - 不自觉地后退半步<br>- 用奇怪的眼神看对方<br>- 内心独白："这个人……不对劲"<br>- 保持距离 |
| **"想保护她"** | - 下意识地站在她身前<br>- 警惕地看着任何靠近她的人<br>- 内心独白："我不能让她出事"<br>- 找借口和她待在一起 |
| **"他在算什么"** | - 警惕地观察他的一举一动<br>- 内心独白："这家伙……在打什么算盘"<br>- 不接他的话/不跟他单独待着 |

---

## 具体例子

### 例子1：周阳看陈默 → 直觉："他知道什么"

❌ **错误写法（知道太多）：**
> "陈默是劫主，他保留轮回记忆。他在笔记本上记录我的行为。"

✅ **正确写法（只有直觉）：**
> 我看到陈默走过来。我下意识地把右手插得更深了。他看了我一眼。那眼神……不对。不是看陌生人的眼神。是看……什么东西的眼神。他知道什么。但他什么都没说。只是点了点头，走过去了。我站在原地，手心全是汗。

✅ **另一种写法（内心独白）：**
> （内心独白）他又在记笔记了。他在写什么？为什么每次看我的眼神都……像见过我很多次？不对。不可能。我们今天才第一次见。但就是……不对劲。我赶紧转过身，假装看窗外。不能让他看出我在怕他。

---

### 例子2：周阳看方衡 → 直觉："他能看穿我。怕他。"

✅ **正确写法：**
> 方衡走过来了。我突然觉得……头有点痛。不是感冒那种痛。是……被什么东西扫过的感觉。他笑了笑，说："周阳是吧？训练辛苦了。"但他的眼睛没笑。那眼神像在看……一个犯人。我赶紧低下头，含糊地嗯了一声。等他走过去，我才发现后背全湿了。

---

### 例子3：周阳看唐小棠 → 直觉："同类。她也在装。"

✅ **正确写法：**
> 走廊转角，我和唐小棠偶遇。我们对视了一秒。就一秒。然后她立刻低下头，脸红了，像个害羞的学妹。但那一秒……我看到了。她眼睛里的东西。不是害羞。是……和我一样的东西。在伪装下面的东西。她也在装。我们擦肩而过，没有说话。但我们都知道了。对方在装。

---

## 三层写法参考

### 第一层：表面行为（最安全，所有人都能看到）
> 我看到方衡，笑了笑，点了点头。然后继续打球。

### 第二层：微动作（只有细心的人能注意到）
> 我看到方衡，笑了笑，点了点头。但右手在口袋里握得更紧了。然后继续打球。

### 第三层：内心独白（只有你知道）
> 我看到方衡，笑了笑，点了点头。但右手在口袋里握得更紧了。（内心独白）他能看穿我。刚才那一下头痛……是他弄的。他到底是什么人？不能让他发现。不能让任何人发现。我用力挥拍，把球打飞出去。

---

## 最后提醒

1. **永远不要解释"为什么"有这种直觉**——你（角色）不知道为什么。你就是……有这种感觉。
2. **用"动作/微表情/身体反应"来表现，而不是直接说"我觉得他不对劲"**
3. **模糊是好的**——越模糊，越有神秘感，越恐慌。
