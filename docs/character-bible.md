# 角色圣经（作者用）

本文档包含所有角色的完整设定，供作者写作时参考。**玩家在游戏中不会直接看到这些信息**，而是通过剧情、线索和陈默的推理逐步解锁。

玩家可见的渐进式人物面板在 `src/passages/common/Common_Characters.twee`。

---

## 陈默

- **年龄**：22
- **专业/身份**：信息工程学院计算机科学专业大四学生
- **表面信息**：冷静理性，寡言，习惯用笔记与逻辑拆解问题
- **与男主关系**：本人
- **隐藏身份**：无限记忆者（特殊村民）
- **能力**：跨轮回保留完整记忆与随身物品状态
- **特长**：逻辑分析、记笔记、观察系统漏洞
- **黑历史**：曾见死不救
- **写作要点**：无限流核心，所有线索的最终整合者。可以比其他人更早发现规则漏洞。

## 苏晚

- **年龄**：21
- **专业/身份**：医学院临床医学专业大四学生
- **表面信息**：温柔但倔强，有医者本能
- **与男主关系**：陈默的恋人，本作女主
- **隐藏身份**：村民
- **能力**：无超自然能力
- **特长**：急救、诊断中毒/伤势、安抚他人
- **黑历史**：无
- **写作要点**：正面角色，陈默的情感锚点。她不会主动作恶，是“干净”的参照系。

## 江白

- **年龄**：22
- **专业/身份**：电气电子工程学院电气工程专业大三学生
- **表面信息**：踏实可靠，话少但动手能力强
- **与男主关系**：陈默的室友，死党
- **隐藏身份**：村民
- **能力**：无超自然能力
- **特长**：修电路、接发电机、做简单工具
- **黑历史**：无
- **写作要点**：普通大学生的义气代表。他的“无黑历史”与陈默、老郑形成对比。

## 方衡

- **年龄**：35
- **专业/身份**：市刑警支队副队长
- **表面信息**：表面温和，习惯观察微表情
- **与男主关系**：陌生人
- **隐藏身份**：预言家 / 黑警
- **能力**：探知对方是盟友还是敌人；村民怀有杀意时探测为敌人
- **特长**：读微表情、审讯、栽赃、追踪
- **黑历史**：黑警，擅长把真相导向自己想要的方向
- **写作要点**：与赵明城相识，此行另有目的。可作为信息源，也可作为误导源。

## 叶知秋

- **年龄**：32
- **专业/身份**：前医学院学生，现为殡仪馆入殓师
- **表面信息**：沉静温柔，常年与尸体打交道
- **与男主关系**：陌生人
- **隐藏身份**：女巫
- **能力**：接触物品可下毒；接触尸体可复活
- **特长**：验尸、保存尸体证据、配药/下毒
- **黑历史**：实习期间医疗事故致人死亡，被退学后转做入殓师
- **写作要点**：“能救人也能害人”的典型。她的医疗背景是能力来源，也是道德污点。

## 沈慎

- **年龄**：38
- **专业/身份**：地下赌场荷官（表面），兼老千（隐藏）
- **表面信息**：沉默寡言，动作利落，总坐在角落
- **与男主关系**：陌生人
- **隐藏身份**：魔术师（致幻）
- **能力**：使狼人在袭击时产生幻觉，误杀同伴
- **特长**：出千、致幻布置、冷读术
- **黑历史**：出千导致一人家破人亡
- **写作要点**：赌徒心态，习惯用欺骗控制局面。致幻能力对应“操控感知”。

## 郑守山 / 老郑

- **年龄**：58
- **专业/身份**：退休旅行团导游
- **表面信息**：独眼，跛足，自称熟悉山庄
- **与男主关系**：陌生人
- **隐藏身份**：村民 / 上一轮轮回幸存者
- **能力**：无超自然能力
- **特长**：熟悉山庄结构、野外生存、隐藏通道
- **黑历史**：上一轮幸存者，知道部分真相但不敢多说
- **写作要点**：信息优势角色，但信息有条件才透露。他的“幸存”本身就可成为线索。

## 林小满

- **年龄**：20
- **专业/身份**：艺术学院绘画专业学生
- **表面信息**：活泼仗义，唐小棠同校闺蜜，有武力值
- **与男主关系**：陌生人
- **隐藏身份**：骑士
- **能力**：私下接触时可发动“决斗”斩杀狼人；对普通村民发动则自己死亡
- **特长**：打架、保护他人、观察色彩/细节
- **黑历史**：小太妹背景
- **写作要点**：护短、义气。与唐小棠的闺蜜关系是她的核心驱动力，也是潜在悲剧线。

## 周阳

- **年龄**：23
- **专业/身份**：体育学院田径专业研究生
- **表面信息**：表面开朗乐观，高大健壮
- **与男主关系**：陌生人
- **隐藏身份**：狼王
- **能力**：被村民或神职杀死时可同归于尽；被女巫下毒时无法发动
- **特长**：奔跑/追逐、体能压制、煽动气氛
- **黑历史**：好胜心强， charismatic 的捕食者
- **写作要点**：狼队天然领袖。阳光人设与狼身份的反差是重要戏剧点。

## 唐小棠

- **年龄**：20
- **专业/身份**：文学院汉语言文学专业大二学生
- **表面信息**：活泼可爱，陈默学妹，精神脆弱
- **与男主关系**：学妹，暗恋陈默
- **隐藏身份**：隐狼
- **能力**：预言家探测时显示为“盟友”
- **特长**：读文字/暗号、察言观色、情感操控
- **黑历史**：幼年遭养父虐待猥亵，黑化后精神状态不稳
- **写作要点**：最危险的“可爱”伪装。她对陈默的感情与狼身份形成巨大张力。

## 赵明城

- **年龄**：45
- **专业/身份**：外贸公司老板，政治掮客
- **表面信息**：圆滑世故，擅长斡旋
- **与男主关系**：陌生人
- **隐藏身份**：清道夫狼
- **能力**：击杀后可以抹除尸体，女巫无法复活
- **特长**：谈判、找把柄、资源调配、心理施压
- **黑历史**：政治家的黑手套
- **写作要点**：与方衡相识，此行另有私事。狼队中的“谋士”角色。

## 顾言

- **年龄**：24
- **专业/身份**：理学院物理学博士研究生
- **表面信息**：敏感，爱观察人，容易崩溃
- **与男主关系**：陌生人
- **隐藏身份**：机械狼
- **能力**：亲手弑神后获得该神职能力
- **特长**：物理推理、拆解机关、实验验证
- **黑历史**：强夺同门师兄弟的学术成果
- **写作要点**：渴望证明自己。弑神夺能力是他在狼队中的独特价值。

## 人物关系简图

- 陈默 ↔ 苏晚：恋人
- 陈默 ↔ 江白：室友/死党
- 陈默 ↔ 唐小棠：学长/学妹（唐暗恋陈）
- 林小满 ↔ 唐小棠：同校闺蜜
- 方衡 ↔ 赵明城：相识，隐线
- 狼队：周阳、唐小棠、赵明城、顾言
- 神职：方衡、沈慎、叶知秋、林小满
- 村民：陈默、苏晚、江白、老郑

## 写作提示

1. 玩家在 `Common_Characters.twee` 中只能看到表面信息。
2. 隐藏信息通过 `Game.revealInfo(charId, key)` 解锁，例如：
   ```twee
   <<run Game.revealInfo("zhou_yang", "wolf_king")>>
   ```
3. 已解锁信息在人物面板中用 `<<if Game.hasRevealed("zhou_yang", "wolf_king")>>` 判断显示。
4. 不要让角色在剧情中主动说出自己的隐藏身份，除非有系统惩罚或特殊剧情需要。
