## 一、主角团（三人）

### 1.1 陈默 · 劫主

**现代身份**
- 姓名：陈默
- 年龄：22
- 专业：信息工程学院计算机科学专业大四
- 社会阵营：光鲜（学生）

**表面特质**
- 冷静理性，寡言
- 习惯用笔记与逻辑拆解问题
- 观察力强，善于发现系统漏洞

**实验身份**
- 劫主
- 唯一保留轮回记忆的人
- 意识结构特殊，无法被完全回滚

**黑历史**
- 曾见死不救（具体场景待定）

**核心能力**
- 跨轮回保留完整记忆与随身物品状态
- "分析"是玩家的能力，不是角色的技能

**核心恐惧**
- 如果破劫成功，大家获救，但只有自己记得轮回真相
- 如果记忆是系统植入的"虚假记忆"
- 苏晚、江白是否在真实世界存在

**人际关系**
- 苏晚：恋人，情感锚点
- 江白：室友/死党，互相信任
- 唐小棠：学妹，唐暗恋陈，陈可能察觉

**为什么在大巴上**
- 全国高校文学节活动，学校在邀请名单上。社长拿到名额但临时有事去不了。陈默是社团资深社员被拉上带队——拉上了室友江白，苏晚正好有空也跟着来了。

**写作要点**
- 无限流核心，所有线索的最终整合者
- 玩家通过他的视角收集信息、做出选择
- 他的"保留记忆"不是超能力，更像一种"诅咒"——只有他记得所有人死过

### 1.2 苏晚 · 凡人

**现代身份**
- 姓名：苏晚
- 年龄：21
- 专业：医学院临床医学专业大三（成绩优秀）
- 社会阵营：光鲜（学生）

**表面特质**
- 温柔但倔强，有医者本能
- 无黑历史，"干净"的参照系

**实验身份**
- 凡人
- 无神力
- 但至关重要：凡人数量决定伪神力量强弱

**核心能力**
- 急救+基础诊断（学生水平，不是专家）
  - 急救：心肺复苏、止血包扎、处理外伤（教科书水平）
  - 基础诊断：判断死因（"像中毒""是刀伤"），但不精准
  - 不会：配药、手术、解毒（和叶知秋的区别）

**核心作用**
- 陈默的情感锚点
- 凡人数量决定伪神力量强弱
- 和叶知秋形成镜像——一个救人，一个害人

**核心恐惧**
- 发现陈默的"预知"是真的——这意味着她死过无数次

**人际关系**
- 陈默：恋人，信任但质疑他的手段
- 江白：朋友，互相尊重
- 唐小棠：警惕，察觉唐对陈默的感情

**为什么在大巴上**
- 陪男朋友陈默去

**写作要点**
- 正面角色，不会主动作恶
- 她的"不选择"哲学 vs 叶知秋的"选择"哲学

### 1.3 江白 · 凡人

**现代身份**
- 姓名：江白
- 年龄：22
- 专业：电气电子工程学院电气工程专业大三
- 社会阵营：光鲜（学生）
- 天津人

**表面特质**
- 大大咧咧，嘴贫但心细
- 踏实可靠，动手能力强
- 无黑历史，"干净"的参照系

**实验身份**
- 凡人
- 无神力

**核心能力**
- 修电路+简易装置
  - 修电路：山庄断电、电路损坏，他能修
  - 简易装置：用电线、铁丝、电池做陷阱或警报
  - 工科实操经验，动手快但毛躁

**核心作用**
- 陈默的技术支持
- 凡人数量决定伪神力量强弱
- 天津式社交是隐藏优势——别人不防着他，无意中透露信息

**为什么在大巴上**
- 被陈默拽去（"免费的"就来了）

**写作要点**
- 普通大学生的义气代表
- 他和陈默的默契是"不用说就懂"的那种

## 二、伪神阵营（四人）

### 2.1 方衡 · 昭判

**现代身份**
- 姓名：方衡
- 年龄：41
- 职业：对外自称"警察，在市局工作"。实际是市刑警支队副队长、副处级（黑警）
- 社会阵营：光鲜

**表面特质**
- 沉稳敏锐，表面温和
- 习惯观察微表情
- 双手背在身后

**真实特质**
- 黑警，收钱封口
- 和赵明城有旧关系，互相利用

**黑历史**
- 收钱封口，做过脏事

**实验身份**
- 昭判（预言家）
- 神力：明鉴——每晚查验一人，得到金水三档（友善/中立/恶意）
- 查验驱动：不靠"感知神力"，而是靠不安因素——白天行为异常的人让他想查
- **猎人能力（枪·2发）**：随身携带手枪（黑警习惯）
  - 被动反杀：若夜间被堕仙杀死，临死可开枪带走一人
  - 主动击杀：白天可私下击杀高度可疑目标，但枪声暴露位置，可疑度飙升至顶点
  - 子弹仅2发，他精明不会轻举妄动

**现实技能**
- 刑侦直觉——经验丰富的刑警，直觉都较为准确。不是那种有证据的判断，而是直觉层面的感知
- 射击——随身携带手枪（详见猎人能力）

**核心矛盾**
- 黑警身份 vs 伪神"维护秩序"的职责
- 他想除掉赵明城（两人做过交易，赵明城手里有他的把柄。留着是隐患，除掉又可能暴露自己）

**为什么在大巴上**
- 家人在度假村，黑警私事耽误

**人际关系**
- 赵明城：旧识，做过交易，**互相知道对方真实身份**（方衡知道赵明城是白手套，赵明城知道方衡是黑警）。互相牵制——谁先揭发对方等于暴露自己
- 陈默：观察，判断是否有利用价值

**写作要点**
- 黑警身份绝对不能对外暴露
- 第一天主动亮明身份"我是警察，在市局工作"——危机时刻亮明身份能稳定局面、确立权威。但不提级别和部门，只说"市局"
- 开枪后会暴露"有枪"这件事——在中国，能合法持枪的基本只有警察，而方衡是唯一自称警察的人
- 他的刑侦直觉让他成为最强的信息收集者
- 金水/银水信任系统是核心博弈机制

### 2.2 叶知秋 · 渡君

**现代身份**
- 姓名：叶知秋
- 年龄：35
- 职业：入殓师（被开除的医学生）
- 社会阵营：底层

**表面特质**
- 沉静温柔，对生死淡漠
- 话不多，但说出来的话很有分量

**真实特质**
- 曾经是医学生，被开除后做了入殓师
- 入殓师也需要缝补伤口，见得多了，也就知道死者的伤口是什么情况

**黑历史**
- 被开除的医学生（具体原因待定，和医疗事故相关）

**实验身份**
- 渡君（女巫）
- 神力：还魂（感知生命流逝，接触尸体复活）+ 缚魂（阻止目标行动的诅咒）
- **缚魂**：无消耗、无冷却，每晚1次（诅咒不耗精神），在堕仙杀人之前决定
- **还魂**：最多2次，在堕仙杀人之后决定
- **一晚只能行动一次**：缚魂或还魂二选一
- 缚魂命中堕仙 → 该堕仙缺席夜间集会 → 容易触发平安夜
- 温柔不挑人，哪怕救的是堕仙
- **银水保护（v9.5）**：当她本轮还魂救下的人（银水对象）被质疑、即将流放时，她可以公开摊牌保人——以暴露渡君身份为代价，3 票反对强行阻断流放。暴露后狼队下一晚将她列为最高优先级目标。详见设计文档 §8.5.10。

**现实技能**
- 配药：可制作麻痹药/安眠药（不能制作杀人毒药）
- 辨伤：入殓师经验，可观察死者伤口判断死因

**核心矛盾**
- 温柔人设 vs 拥有杀人能力（缚魂）
- 救人一视同仁，但被救的人可能是堕仙

**为什么在大巴上**
- 找老朋友（度假村员工）

**人际关系**
- 苏晚：镜像关系（一个在读医学生，一个退学入殓师）
- 方衡：对方衡和沈慎的印象不好

**写作要点**
- 缚魂无消耗（诅咒层面），还魂最多2次（神力有限）
- 治愈已从神力移除，改为现实技能（需材料）
- 温柔人设：缚魂优先级低，还魂一视同仁
- 还魂2次用完，她不再有救人能力（但缚魂仍可用）

### 2.3 林小满 · 镇煞

**现代身份**
- 姓名：林小满
- 年龄：21
- 专业：艺术学院绘画专业
- 社会阵营：底层（小太妹）
- 形象：金发（染发），小太妹

**表面特质**
- 冲动仗义，护短
- 嘴上不饶人，但心是软的
- 唐小棠的闺蜜，拼命保护她

**真实特质**
- 年幼时保护唐小棠不受同学欺负，要和别人打架
- 学跆拳道的动机：保护唐小棠
- 不知道唐小棠被养父虐待的事

**黑历史**
- 无（她的"太妹"形象是保护唐小棠的副产品，不是真正的坏人）

**实验身份**
- 镇煞（骑士）
- 神力：决斗（斩杀堕仙，对凡人发动则自己死）+ 守卫（保护一人过夜）
- 决斗和守卫不共享冷却——白天决斗，晚上可以守卫
- 守卫可连续守人（中间隔了一天）

**现实技能**
- 跆拳道——物理保护、威慑、近身格斗

**核心矛盾**
- 偏信隐狼唐小棠（闺蜜滤镜），不完全听预言家
- 如果发现唐小棠是堕仙，会崩溃

**为什么在大巴上**
- 陪闺蜜唐小棠去文学节，不是社团成员

**人际关系**
- 唐小棠：闺蜜，拼命保护
- 陈默：临时同盟
- 苏晚：友好

**写作要点**
- 她的跆拳道不是为了伤害别人，是为了保护唐小棠
- 唐小棠是她最重要的精神支柱——发现真相会动摇她的全部信念

### 2.4 沈慎 · 幻真

**现代身份**
- 姓名：沈慎
- 年龄：45
- 职业：度假村网管（表面），度假村地下赌场荷官（隐藏）
- 社会阵营：底层

**表面特质**
- 沉默寡言，总坐在角落
- 动作利落，手指灵活
- 不起眼，容易被忽视

**真实特质**
- 赌场荷官，出千高手
- 曾因出千害得债主家破人亡
- 操控欲强，把所有人当"筹码"

**黑历史**
- 出千害得人家破人亡（不是直接杀人，是间接毁灭）

**实验身份**
- 幻真（魔术师）
- 神力：镜花——**每晚必定交换两人**（赌徒妄想症，不交换睡不着）
- 可交换任何人，包括自己
- 只有堕仙形态能看见交换
- 不能连续两晚交换同一对人

**自保模式**（觉得危险或30%恐慌）：把自己和"安全的人"交换
  - 猜对 → 替死鬼死，沈慎活
  - 猜错 → 沈慎自己死（赌输了）

**赌博模式**（觉得安全）：交换"可能被盯上的人" ↔ "可疑的人"
  - 猜对 → 堕仙误杀同伴
  - 猜错 → 无辜的人替死

**现实技能**
- 手法——赌场荷官的出千、偷牌、调包。可以制造"物品丢失/被调换"的剧情场景

**核心矛盾**
- 自私的赌徒：只为苟活，不主动收集/分享情报
- 妄想症让他每晚都必须交换，但交换本身就是赌博——猜错就死
- 他想低调，但镜花的力量逼他每晚下注

**为什么在大巴上**
- 休假结束，从公交站坐大巴回度假村上班

**人际关系**
- 赵明城：同行（黑白两道），互相试探
- 陈默：观察，试图定价

**写作要点**
- 赌场荷官身份不能对外暴露
- 他的"手法"是现实技能，"镜花"是神力
- **沈慎是自私的赌徒**：只为苟活，行事低调，不主动分享情报
- 他的交换是赌注——有时救了别人（误杀同伴），有时害死自己（猜错）

## 三、堕仙阵营（四人）

### 3.1 周阳 · 幽主

**现代身份**
- 姓名：周阳
- 年龄：23
- 专业：体育学院田径专业研究生
- 社会阵营：底层

**表面特质**
- 阳光开朗，高大健壮
- 人缘好，有号召力
- 体育生的活力

**真实特质**
- 训练过度，身上有伤
- 医院建议温泉疗养——这是他来度假村的理由
- 阳光外表下有暴力倾向的底色（不是体育竞技的攻击性，是更深的东西）
- 右手总插口袋，握成拳头——在压抑那种想终结一切的冲动
- 23:00之后会灵魂饿——不是胃饿，是一种无法解释的渴望

**黑历史**
- 小时候被霸凌。反杀了领头的。转学了
- 这段记忆被他压在最深处。阳光是装出来的，装太久了，连他自己都信了
- 系统激活后，那种冲动最先被释放——他的杀心是三狼中最高的

**实验身份**
- 幽主（狼王）
- 神力：同归于尽——被凡人/伪神杀死时可带走杀死他的人
- 被缚魂不能发动，被毒（麻痹/安眠）仍可发动

**现实技能**
- 体能——追击、压制、体力优势

**核心矛盾**
- 阳光人设 vs 堕仙身份
- 白天和大家打球笑闹，晚上被系统驱使狩猎

**为什么在大巴上**
- 训练过度，身上有伤，医院建议温泉疗养

**人际关系**
- 陈默："特殊"的猎物
- 林小满：敌意，但察觉她的"被迫"
- 唐小棠：堕仙同伴

**写作要点**
- 堕仙天然领袖
- 阳光人设与堕仙身份的反差是重要戏剧点
- 不是天生残忍，但他离残忍最近——小时候的暴力记忆是系统侵蚀他的入口
- 他的杀心递增最快（Day2就可能动手）

### 3.2 唐小棠 · 匿魂

**现代身份**
- 姓名：唐小棠
- 年龄：20
- 专业：文学院汉语言文学专业大二
- 社会阵营：光鲜（学生）

**表面特质**
- 活泼可爱，小鸟依人
- 陈默的学妹，文学社成员
- 表面乐观合群

**真实特质**
- 幼年遭养父虐待猥亵
- 因为被虐待导致性格孤僻
- 后来有林小满陪伴，走出阴影，开始变得乐观合群——但这个"走出来"是表面的，幼年创伤没有真正愈合
- 她的伪装不是技能，是生存方式

**黑历史**
- 幼年遭养父虐待猥亵（她从来没告诉过任何人，包括林小满）

**实验身份**
- 匿魂（隐狼）
- 神力：隐匿——预言家（方衡）查验时始终显示为"友善"或"中立"，验不出来

**现实技能**
- 伪装——无法轻易被看破。幼年创伤让她学会了隐藏自己

**核心矛盾**

## 七、破局核心

> 系统的规则是让伪神和堕仙互相残杀，但真正的破局方法是——大家团结。

顾言（噬道·机械狼）的"解析"能力是破局的关键拼图。他意识不到系统的本质，需要陈默（玩家）用轮回经验引导他觉醒。

- 顾言活着 + 玩家引导他觉醒 → 可以达成真结局
- 顾言死了 → 玩家失去解析能力，**达不到真结局**
- 真结局需要全员存活（包括所有堕仙）
- 这是玩家的核心道德选择：顾言是堕仙，但保护他、信任他、说服他放弃夺舍转向合作——还是让他作为堕仙被淘汰？

## 八、写作提示

1. **玩家在 Common_Characters.twee 中只能看到表面信息**
2. **隐藏信息通过 Game.revealInfo(charId, key) 解锁**
3. **已解锁信息在人物面板中用 Game.hasRevealed 判断显示**
4. **不要让角色在剧情中主动说出自己的隐藏身份**，除非有系统惩罚或特殊剧情需要
5. **双重错位**：社会阵营（光鲜/底层）≠ 实验身份（伪神/堕仙/凡人）≠ 善恶
6. **政治掮客/白手套身份绝对不能对外暴露**，赵明城对外只是"外贸公司老板"
7. **赌场荷官身份不能对外暴露**，沈慎对外只是"度假村网管"
8. **黑警身份不能对外暴露**，方衡对外只是"刑警副队长"
9. **角色不应该预知自己会死或会杀人**（除非是系统宣告）
10. **不要标签化内心独白**（"棋子"、"炮灰"、"敌人"、"变态"等）
11. **陈默是玩家角色**，必须是"白纸"——他的文档只保留基础信息
12. **女巫温柔，不戴有色眼镜看人**
13. **故事本质不是狼人杀**，是关于团结求生的无限流
14. **神力的真假保持暧昧**，不要在剧情里坐实超自然现象
