# 《无尽雪夜》游戏设计文档 v9

> 版本：v9
> 更新日期：2026-06-21
> 核心设定：十二名真实遇难者的意识被困于劫墟，七日轮回，濒死情感唤醒系统

---

## 一、世界观

### 1.1 真实事件

2026年冬，一辆大巴从市区公交站发车，前往落雁山庄度假村。车上有学生、商人、警察、入殓师、体育生、赌场荷官、退休司机——十二个互不相识的人。

途中遭遇暴风雪，盘山公路发生雪崩，大巴坠崖。
十二人全部遇难。肉体在雪地里逐渐变冷，生命体征微弱到几乎消失。

### 1.2 劫墟的本质

落雁山庄是**真实存在的地点**。但十二人在劫墟中经历的"七日轮回"，发生在他们的**意识空间**里。

劫墟不是某个高高在上的存在创造的实验场。它是被**十二个人濒死时的绝望与求生欲**唤醒的——他们不想死，这个念头太强了，强到在意识的边界凝聚出了一个空间。

系统按照自己的逻辑运行：让"伪神"（神职）和"堕仙"（狼）互相残杀，活下来的人就是"值得被救的人"。这是系统对"求生"的扭曲理解——它以为淘汰弱者就是帮助大家活下去。

但系统的规则不是真理。真正的破局方法，是**团结**。

### 1.3 关于"神力"的暧昧性（重要设计哲学）

游戏里，神职和堕仙的能力被包装成"神力"。但这些能力到底是真是假，玩家可以自己判断——**这是故意留下的遐想空间**。

- 叶知秋的"还魂"，在劫墟里是真的复活；现实里可能就是那个人心跳恢复、呼吸重启
- 方衡的"明鉴"，可能只是刑警的职业直觉被濒死意识放大了
- 沈慎的"镜花"，可能只是赌徒对人性弱点的精准利用被外化了
- 林小满的"镇煞"，可能只是她保护闺蜜的本能被赋予了名字

不要在文档或剧情里把神力"坐实"为超自然现象。让玩家自己判断：这到底是濒死意识里的挣扎，还是真的有某种力量在运作？

## 二、阵营体系

### 2.1 三大阵营

| 阵营 | 命名 | 人数 | 核心特征 |
|------|------|------|----------|
| 守序者 | **伪神** | 4人 | 被系统授予神力，力量来源于凡人的信任 |
| 破序者 | **堕仙** | 4人 | 被系统授予异能，力量来源于夜光，只能夜晚行动 |
| 无能力者 | **凡人** | 4人 | 无神力。陈默（劫主·轮回记忆）、苏晚、江白、老郑 |

### 2.2 阵营核心矛盾

- **伪神**以为自己是"神"，拥有力量，实际是系统分配的角色
- **堕仙**以为自己是"反抗者"或"猎人"，实际也是系统的棋子
- **凡人**以为自己是普通人——陈默除外，他是唯一知道"轮回存在"的人，但他的记忆也可能被系统篡改

### 2.3 信息边界（最重要的规则）

- **所有角色只知道**：自己的身份和技能、白天发生的事、线索的原始内容
- **堕仙第一夜碰面**：知道彼此是堕仙，但不互相告知具体能力
- **堕仙不知道伪神的身份和技能**
- **堕仙之间也不知道彼此的技能**
- **伪神之间互不知晓身份和技能**
- **所有人都不知道**：其他人的身份和技能（除非公开、被查验、或被夺舍）

## 四、核心机制

### 4.1 神力 vs 现实技能

| | 神力/特殊能力 | 现实技能 |
|---|---|---|
| 来源 | 系统授予 | 角色自身经历 |
| 堕仙可见？ | 否（连队友的都看不到） | 白天行为可观察 |
| 消耗 | 有次数/冷却限制 | 无消耗，但需要场景条件 |
| 举例 | 方衡·明鉴、叶知秋·还魂 | 方衡·刑侦直觉、叶知秋·配药 |

### 4.2 行动时机

- **伪神**：任意时间发动神力（力量来源于凡人的信任）
- **堕仙**：只能夜晚发动能力（力量来源于夜光）
- **例外**：
  - 幽主（周阳）白天被杀时可发动同归于尽
  - 噬道（顾言）白天也可使用偷来的能力（一次性）

### 4.3 凡人

凡人（陈默、苏晚、江白、老郑）无神力。他们是这场博弈中最普通的参与者，但：
- 陈默（劫主）拥有轮回记忆，是唯一的变量
- 其余三人完全不知道轮回的存在，每轮重置

## 五、技能详细规则

### 5.1 伪神

#### 方衡 · 昭判 · 明鉴（预言家）

- **每晚查验一人**，得到**金水三档**：友善 / 中立 / 恶意
- 查验结果只代表"对方对方衡的态度/威胁性"，**不是**"好人/坏人"
  - 匿魂（唐小棠）始终验为**友善或中立**——验不出来
  - 陈默始终验为**中立**（系统不暴露他）
  - 伪神也可能验为"恶意"（如果对方对方衡有敌意）
- **查验驱动**：方衡不靠"感知神力"，而是靠**不安因素**——白天行为异常、让他警觉的人，他会优先查验
- **金水/银水**：方衡查验过的人，他可以对外声称是"金水"（验过，态度友善）；叶知秋救过的人，称为"银水"
- **堕仙看不到方衡的查验结果**。查验是方衡的内心判断，只有他说出来别人才知道
- 方衡不知道自己救没救过谁（救人由女巫负责）

#### 叶知秋 · 渡君 · 还魂 + 缚魂（女巫）

- **还魂**：感知"今晚有人死去"，可前往救活（不暴露身份，死者昏迷不知情）
- **缚魂**：阻止某人今晚的一切行动（伪神和堕仙都失效）。缚魂不杀人，是"诅咒"层面的束缚
- **配药**（现实技能）：可制作**麻痹药 / 安眠药**，给某人下药，该人今晚丧失行动能力
- **辨伤**（现实技能）：入殓师经验，可观察死者伤口判断死因
- **次数限制**：还魂 + 缚魂**合计不超过 3 次**，否则精神崩溃
- **每天只能行动一次**：今晚要么还魂，要么缚魂，不能同时
- **女巫温柔，不戴有色眼镜看人**：救人不挑对象，哪怕救的是堕仙
- **女巫不知道堕仙的动向**：缚魂/还魂的决定必须在堕仙行动之前做出，只能根据白天线索判断
- **治愈已从神力中移除**：重伤治疗是现实技能（需材料），不在 3 次神力额度内

#### 林小满 · 镇煞 · 决斗 + 守卫（骑士）

- **决斗**：白天私下接触时可发动，斩杀堕仙。对凡人发动则**自己死亡**
- **守卫**：保护一人过夜，该人今晚不会被堕仙杀死。不能守自己
- **冷却**：决斗和守卫**不共享冷却**——白天决斗，晚上可以守卫
- **守卫可连续守人**：中间隔了一天，所以"今晚守 A，明晚守 A"是允许的
- **同守同救必死**：骑士守卫 + 女巫还魂同时作用于一人，保护失效，目标死亡
- **骑士不完全听预言家**：骑士性格冲动仗义，她**偏信隐狼唐小棠**（闺蜜滤镜），不会对预言家言听计从

#### 沈慎 · 幻真 · 镜花（魔术师）

- **每晚选择两人交换身份**：堕仙袭击 A 时，实际杀死的是 B；堕仙以为杀了目标，其实杀了同伴
- **只有堕仙形态能看见交换**：白天大家都不知道发生了什么
- **不能连续两晚交换同一对人**：上一晚参与交换的两人，下一晚不能再被选中
- **沈慎每晚的交换是秘密的**：堕仙无法推测他是否交换了、交换了谁
- **现实技能·手法**：赌场荷官的出千、偷牌、调包——可以制造"物品丢失/被调换"的剧情场景

### 5.2 堕仙

#### 周阳 · 幽主 · 同归于尽（狼王）

- 白天或夜晚被凡人/伪神杀死时，可发动同归于尽，带走杀死他的人
- **被缚魂（女巫诅咒）不能发动**
- **被毒（现实药剂）也能发动**（毒只是麻痹/安眠，不致死）
- 只有被"直接击杀"才触发

#### 唐小棠 · 匿魂 · 隐匿（隐狼）

- 预言家（方衡）查验时**始终显示为友善或中立**——验不出来
- **唐小棠杀心不重**：暗恋陈默，情绪化，不会主动带节奏杀人
- 堕仙讨论时，**唐小棠不引导**（不提议杀谁），只表态同意/反对

#### 赵明城 · 无常 · 抹尸（清道夫狼）

- 亲手杀人后可抹除尸体：女巫（叶知秋）**无法感知、无法还魂**
- **抹尸只针对伪神才触发**：杀的人是不是伪神，杀之前谁都不知道。只有杀完，且此人是伪神，才会触发抹尸效果。杀凡人不触发
- **首要目标是方衡**：赵明城和方衡有隐线（互相知道对方秘密），他此行有私事，最想除掉方衡
- **对老郑无特别动机**
- **现实技能·斡旋**：调解矛盾、拉拢离间

#### 顾言 · 噬道 · 夺舍（机械狼）

- **必须亲手杀伪神**才能夺舍：队友杀的不算
- 夺舍后获得该伪神能力的**变体**，偷来的技能**只能用一次**：
  - 偷昭判 → 可直接查看对方身份（比金水三档更强）
  - 偷渡君 → 只能毒 1 次（毒药版本，杀人用）
  - 偷镇煞 → 决斗，但**逻辑相反**（对堕仙发动自己死，对凡人发动成功）——可以白天"决斗"杀人，自称镇煞
  - 偷幻真 → 交换身份（和原版一样）
- **现实技能·解析**：理解系统规则、发现漏洞。唯一能从科学角度分析山庄规则的人。**破局关键**——他死了，玩家失去理解规则的能力，达不到真结局
- 顾言意识不到系统的本质（团结才是破局），需要陈默用轮回经验引导他觉醒

## 六、夜晚行动顺序

一个夜晚的完整流程：

1. **镇煞（林小满）守卫决策** —— 选择今晚守护谁（白天已决定）
2. **昭判（方衡）查验** —— 根据白天的不安因素，查验一人
3. **渡君（叶知秋）缚魂决策** —— 在堕仙行动前，决定今晚是否缚魂（缚谁）。渡君不知道堕仙会杀谁，只能凭白天线索判断
4. **堕仙讨论 + 杀人** —— 引导者提议 → 表态 → 派人执行
5. **幻真（沈慎）交换** —— 堕仙行动后，交换两人身份（可能让堕仙的击杀打偏）
6. **渡君（叶知秋）还魂决策** —— 感知到有人死去，决定是否还魂

**关键点**：
- 渡君一晚只能行动一次：缚魂（第3步）或还魂（第6步），二选一
- 缚魂在堕仙杀之前，是预防性的；还魂在堕仙杀之后，是补救性的
- 幻真交换在堕仙杀之后——所以幻真可以让堕仙"以为自己杀了目标，实际杀了同伴"
- 同守同救必死：镇煞守卫 + 渡君还魂同时作用于一人，保护失效，目标死亡

## 七、堕仙讨论系统

堕仙不再是加权随机投票，而是**引导者提议 → 其他堕仙表态 → 最多 3 轮**。

### 7.1 引导者优先级

1. 周阳（优先引导）
2. 赵明城（其次）
3. 顾言（按概率，有时引导有时沉默）
4. **唐小棠不引导**（情绪化，不主动带节奏）

### 7.2 讨论流程

- 引导者根据白天观察提出击杀目标，并**阐述理由**
- 其他堕仙表态：同意 / 反对 / 另提人选
- 最多 3 轮。通过则执行；不通过则进入安全夜（当晚不杀人）或由下一提议者接力
- **不通过也可以是安全夜**：如果没有共识，堕仙选择不动手（避免暴露）

### 7.3 白天信息对堕仙的影响

堕仙白天**不统一行动**，不公开交换情报。情报交换必须私下进行。

晚上杀人时，堕仙变异化了，**不能语言沟通，只能最简单的比划**（杀谁、同不同意）。

堕仙根据白天的动态调整杀心：
- 两个伪神吵架 → 杀其中一个，让大家以为是矛盾激化
- 某人表现得太聪明/太接近真相 → 增加杀心
- 栽赃嫁祸：制造"某人像堕仙"的假象

## 八、信任机制

### 8.1 金水系统（昭判→渡君）

- 方衡查验某人后，可以**自主决定告诉谁**
- 方衡告诉渡君，渡君会信（渡君对金水的信任值高）
- 方衡告诉其他人，对方不一定信
- **转述存在风险**：你转述的是真话还是假话？被转述的人无法验证
- 方衡怎么知道谁是渡君？**他不知道**——他只能根据白天的观察、救人的行为来猜测

### 8.2 银水系统（渡君→被救者）

- 渡君救过的人，渡君会信任（银水）
- 渡君也信任方衡告诉她的金水
- 渡君信任自己救过的人 > 信任方衡的金水

### 8.3 镇煞的信任偏向

- 林小满性格冲动仗义，她**偏信隐狼唐小棠**（闺蜜滤镜）
- 她不会对昭判（方衡）言听计从
- 这意味着即使方衡给出了正确的金水，林小满也可能不采纳

## 九、结局系统

结局分为**主轴结局**（和轮回绑定）和**分支结局**（基于角色关系和玩家选择）。

### 9.1 主轴结局

| 结局 | 触发条件 | 结果 |
|------|---------|------|
| **坏结局** | Day7 前陈默死亡 | 意识崩溃，重新轮回 |
| **好结局** | Day7 结束时有幸存者 | 活几人 = 现实获救几人，其余牺牲 |
| **真结局** | 集齐所有破局线索 + 全员存活（包括所有堕仙） | 12 人全部获救，系统觉醒 |

### 9.2 分支结局（持续扩展中）

分支结局基于玩家在轮回中的具体选择、角色关系、线索收集情况触发。以下为初步设计，后续可不断添加：

| 结局 | 触发方向 | 备注 |
|------|---------|------|
| 苏晚结局变体 | 苏晚存活/死亡 | 影响好结局的尾声 |
| 顾言结局变体 | 顾言被保护/被淘汰 | 决定能否达到真结局 |
| 唐小棠结局变体 | 唐小棠黑历史被揭发后的走向 | 情感线 |
| 林小满结局变体 | 林小满发现唐小棠真实身份后的选择 | 闺蜜线 |
| 方衡结局变体 | 方衡的黑警身份被揭发 | 道德选择 |
| 沈慎结局变体 | 沈慎的赌场身份被揭发 | 道德选择 |

分支结局的具体触发条件和文本在剧情开发中逐步完善。

### 9.3 真结局与顾言

顾言是破局的关键拼图。他的"解析"能力是唯一能从科学角度理解系统规则的工具。

- 顾言活着 + 玩家引导他觉醒 → 可以达成真结局
- 顾言死了 → 玩家失去解析能力，**达不到真结局**（最多好结局）
- 这是玩家的核心道德选择之一：顾言是堕仙，但保护他、信任他、说服他放弃夺舍转向合作——还是让他作为堕仙被淘汰？

## 十、现实技能与剧情

现实技能不走神力系统，走 **passage 内的选择分支**。它们为剧情服务：

| 角色 | 现实技能 | 可能的剧情用途 |
|------|---------|--------------|
| 陈默 | （玩家能力） | 推理、记笔记、观察 |

## 十一、AI 系统

### 11.1 设计原则

这是**静态游戏**，没有 AI 实时处理。所有"AI 决策"在游戏构建时/轮回开始时**预计算**，存入状态。玩家看到的是预计算的结果。

这意味着：
- 难度在于**预计算的合理性和多样性**
- 同一轮回里，事件序列是固定的（玩家读档/轮回才变化）
- 但不同轮回之间，因为陈默的选择不同，事件可以不同

### 11.2 各角色 AI 要点

| 角色 | AI 要点 |
|------|--------|
| 方衡 | 查验让他"不安"的人（行为异常）；金水三档；可对外声称金水，但转述有风险 |
| 叶知秋 | 温柔不挑人；缚魂优先级低（不爱杀人）；还魂一视同仁（哪怕救的是堕仙）；3 次额度耗尽则精神崩溃 |
| 林小满 | 决斗偏信唐小棠（闺蜜滤镜），不完全听预言家；守卫倾向保护弱势/有冲突的人 |
| 沈慎 | 交换不连续重复；优先交换"可能被堕仙盯上的人"和"堕仙自己"——但沈慎不知道谁是堕仙，靠冷读术猜测 |
| 堕仙 | 引导者提议制；赵明城优先方衡；唐小棠不引导；根据白天动态调整 |

## 十三、写作红线

1. **不随意改写剧情**：本游戏有完整剧本，工程化、结构化、可维护化是首要任务
2. **NPC 不能"知道"其他角色的身份或能力**：所有直觉必须模糊
3. **不要标签化内心独白**（"棋子"、"炮灰"、"敌人"等）
4. **不要预知化**：角色不应该知道自己会死或会杀人（除非是系统宣告）
5. **角色面板不能透露敏感身份**（政治掮客、赌场荷官、黑警等）
6. **陈默是玩家角色**，必须是"白纸"——他的文档只保留基础信息，不写轮回记忆细节
7. **女巫温柔，不戴有色眼镜看人**
8. **故事本质不是狼人杀**，是关于团结求生的无限流
9. **神力的真假保持暧昧**，不要在剧情里坐实超自然现象
